Key Technical Decisions

Database: PostgreSQL with Prisma ORM for type safety
Authentication: JWT tokens with refresh mechanism
File Storage: AWS S3 for all media and generated portfolios
AI Service: Claude API for intelligent content generation
Hosting: AWS Amplify (frontend) + ECS Fargate (backend)
CDN: CloudFront for global distribution of portfolios
